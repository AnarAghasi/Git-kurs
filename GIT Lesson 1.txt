1.	Перейти в созданную папку с помощью команды cd  (например, cd Documents/MyRepo).
2.	Вернутся на GitHub в репозиторий, который мы хотели скопировать. Нажимаем на кнопку "Code" (или "Код") и скопируем URL репозитория.
3.	В командной строке или терминале ввыдим команду  git clone , вставим скопированный URL после git clone. Например, git clone 

2.Как получить на локальный компьютер изменения с github.com
Открываем  командную строку или терминал на нашем компьютере.
Переходим в папку репозитория на локальном компьютере с помощью команды cd . 
Ввыводим команду git pull origin, чтобы получить изменения из удаленного репозитория. Заменим на имя ветки, из которой мы хотим получить изменения. Обычно используется ветка "master" или "main". Например, git pull origin master.
Git выполнит операцию "pull", которая заберет последние изменения с GitHub и автоматически объединит их с вашей локальной версией репозитория.

3,Какую информацию нам дают команда : git status git log git remote -v 
Команда -«git status» предоставляет информацию о текущем состоянии  локального репозитория Git. Она показывает, были ли внесены изменения, добавлены ли новые файлы или удалены.
Команда –«git log» отображает историю коммитов репозитории Git. Она показывает список предыдущих коммитов вместе с информацией о каждом коммите.
Команда-« git remote –v» отображает список удаленных репозиториев, связанных локальным репозиторием Git. Она показывает URL удаленных репозиториев и их имена (обычно "origin"),
