1.Для создания Git-репозитория из локального каталога на своем компьютере и отправки его на GitHub.com следуйте следующим шагам:

Перейдите  GitHub.com, войдите в свою учетную запись и сделать "New repository"
git remote add origin <repository-url> 
с помощью команды git push -u origin master. Это отправит наши коммиты в удаленный репозиторию 

2.Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?

git checkout . для отмены всех изменений во всех файлах в рабочем каталоге.


3.Как вернуть файл из индекса в рабочий каталог?

git restore --staged <file>. Эта команда перемещает изменения, связанные с указанным файлом, из индекса обратно в рабочий каталог.


4.Как отменить ошибочный commit ? 
 
Команда git reset --hard HEAD~<number> для удаления последних <number> коммитов из истории, 
включая изменения, внесенные в них. Замените <number> на количество коммитов, которые мы хотим удалить. 
Например, чтобы удалить последний коммит, выполните следующую команду:
git reset --hard HEAD~1
Это удалит последний коммит и все изменения, связанные с ним, из истории.
